# Single LED Blinking Using a Switch with AT89C51 Microcontroller

## Description
This project demonstrates controlling a single LED using a push button switch with the AT89C51 (8051) microcontroller. When the switch is pressed, the LED turns ON, and when the switch is released, the LED turns OFF. The project is developed using Keil uVision and simulated in Proteus.

## Components Used
- AT89C51 Microcontroller
- Single LED
- Push Button Switch

## Software Used
- Keil uVision
- Proteus 8
- Embedded C

## Files
- single_led.c – Embedded C source code
- single_led.uvproj – Keil project file
- STARTUP.A51 – Startup file
- single_led.pdsprj – Proteus project file
- circuit.cng - circuit diagram
